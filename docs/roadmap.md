# Roadmap

## Phase 1 — Executable contest MVP

- OTA passport
- Explainable anomaly scoring
- Old/new replay diff
- TF-IDF historical case retrieval
- Evidence-bound root-cause hypotheses
- Rule-based counterfactual experiments
- Web demo and report export

## Phase 2 — Real engineering integration

- Connect Git/code/model/config repositories
- Connect object storage and time-series database
- Integrate SIL replay engine
- Add vector database and enterprise RAG
- Add permissions, audit log and data retention

## Phase 3 — Vehicle and fleet intelligence

- Distilled vehicle-side behavior envelope model
- OOD and temporal anomaly models
- Bayesian small-sample fleet drift monitoring
- Hierarchical comparison across vehicle, region and weather

## Phase 4 — End-to-end model diagnosis

- Trajectory distribution divergence
- Input attribution and object influence scoring
- Latent representation drift
- Counterfactual scene editor and sensor simulation

## Phase 5 — Production governance

- Release gates and approval workflow
- Automated regression suite generation
- Evidence package aligned with software-upgrade and event-recording requirements
- Data privacy, desensitization and retention policy
